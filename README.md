## Requirements
* Spring Boot as Backend
* Angular as Frontend

* Spring Web
* JPA
* My SQL

* JWT
### Frontend
* HTML
* CSS
* BOOTSTRAP

* Admin Can add Questions,update questions,delete questions
### User Panel:-
* User can attempt quizzes
